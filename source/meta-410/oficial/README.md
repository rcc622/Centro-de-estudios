# Meta Media Buying Professional 410 — Guías oficiales

Sube aquí las guías oficiales de Meta para el examen **410-101**.

## Qué subir

- `410-guia-oficial-es.pdf` — guía oficial de estudio en **español**
- `410-guia-oficial-en.pdf` — guía oficial de estudio en **inglés**

(Si el PDF pesa > 5 MB no se commitea: Claude lo convierte a `.md` y deja aquí
el `410-guia-oficial-es.md` / `410-guia-oficial-en.md`.)

## Para qué se usa

- **Inglés** → genera el contenido jugable en inglés: `content/meta-410/en/`
- **Español** → alinea los nombres de conceptos clave del español actual a los
  términos oficiales (no traducción automática).

Una vez subidas, avísale a Claude para que las procese.
